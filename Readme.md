# 📝 To-Do List Web App

A simple and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.
This project helps users manage daily tasks by allowing them to **add, delete, and track tasks easily** with a clean and responsive interface.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* 📜 Scrollable task list when tasks increase
* 🎨 Modern and responsive UI
* ⚡ Fast and lightweight (pure HTML, CSS, JavaScript)

---

## 🛠️ Built With

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout design
* **JavaScript (Vanilla JS)** – Functionality and interactivity
